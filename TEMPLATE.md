# Шаблон документа

Это шаблон документа с YAML frontmatter для хранилища знаний SRD.

## YAML Frontmatter

Каждый документ начинается с метаданных в формате YAML:

- **type** — тип документа:
  - `concept` — концептуальное описание
  - `spec` — спецификация
  - `guide` — руководство
  - `decision` — архитектурное решение

- **status** — статус документа:
  - `draft` — черновик
  - `review` — на ревью
  - `active` — активный
  - `archived` — архивный

- **family** — принадлежность к ячейке SRD-матрицы:
  - `"0.1"` — Управление / Логика хранилища
  - `"1.1"` — Надсистема / Смыслы
  - `"1.2"` — Надсистема / Архитектура
  - `"1.3"` — Надсистема / Операции
  - `"2.1"` — Целевая / Смыслы
  - `"2.2"` — Целевая / Архитектура
  - `"2.3"` — Целевая / Операции
  - `"3.1"` — Создания / Смыслы
  - `"3.2"` — Создания / Архитектура
  - `"3.3"` — Создания / Операции

- **created** — дата создания в формате YYYY-MM-DD

## Использование шаблона

1. Скопируй этот файл
2. Замени frontmatter на актуальные значения
3. Заполни содержание документа
4. Размести в соответствующую папку согласно `family`

## Структура содержания

Структура документа произвольная, но рекомендуется:

- Заголовок первого уровня с названием
- Краткое описание назначения
- Основное содержание
- Ссылки на связанные документы (если есть)
