# FPF Index

Индекс спецификаций First Principles Framework для проекта.

## Назначение

Этот каталог содержит спецификации архитектурных решений, основанных на первых принципах (First Principles Framework).

## Структура

- `FPF-Spec.md` — основная спецификация FPF для проекта (создается при необходимости)
- Дополнительные спецификации добавляются по мере развития архитектуры

## Использование

Перед архитектурной работой Claude Code загружает `FPF-Spec.md` для обеспечения согласованности решений.

## Принципы FPF

1. **Разложение** — разложи проблему до базовых элементов
2. **Фундамент** — определи первые принципы и ограничения
3. **Синтез** — собери решение от фундамента вверх
