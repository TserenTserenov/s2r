# 0.5. AI Reports

Автоматические отчёты, генерируемые AI-агентами.

## Типы отчётов

| Тип | Описание | Периодичность |
|-----|----------|---------------|
| Аудит структуры | Проверка целостности хранилища | По запросу |
| Анализ связей | Проверка related-ссылок | Еженедельно |
| Статус заполнения | Какие семейства пусты | По запросу |

## Формат отчёта

```markdown
# [Тип отчёта]

**Дата:** YYYY-MM-DD
**Сгенерировано:** Claude Code

## Резюме

[Краткие выводы]

## Детали

[Подробный анализ]

## Рекомендации

- [ ] Рекомендация 1
- [ ] Рекомендация 2
```
