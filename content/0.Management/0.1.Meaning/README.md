# Смысл управления

## Назначение раздела

Здесь документируются цели, миссия, видение и ценности управления проектом.

## Что здесь размещается

- **Миссия проекта** — для чего существует проект
- **Видение** — желаемое будущее состояние
- **Цели** — измеримые результаты
- **Ценности** — принципы принятия решений
- **Стратегия** — как достигаем целей
